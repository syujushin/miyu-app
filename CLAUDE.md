# Miyu — AI 맞춤형 뷰티 커머스 프로젝트

## 서비스 개요
매일 달라지는 피부 상태와 생활 패턴을 분석하여 사용자에게 꼭 맞는 제품을 찾아주는
AI 맞춤형 뷰티 이커머스 서비스. 핵심 가치는 **개인화, 편의성, 신뢰성**.

브랜드 톤: 글로우 + 고급스러움 + 정제됨 → 전문적이고 투명하며 신뢰도 높은 이미지.
디자인 컨셉: 글래스모피즘, 블루-퍼플 그라디언트, 소프트한 럭셔리 무드.

## 기술 스택
- React + Vite
- Tailwind CSS
- react-router-dom (라우팅)
- 바텀 네비게이션 구조

### 컴포넌트 스타일 가이드
- 글래스모피즘: 반투명 배경(backdrop-blur) + 옅은 보더 + 부드러운 그림자
- 카드: 라운드 코너(넉넉하게, 16~24px), 옅은 그라디언트 배경
- 버튼: 메인 CTA는 `#7733FF` 단색 또는 그라디언트, 보조 버튼은 아웃라인/라이트 톤
- 바텀 네비게이션: 글래스 효과, 아이콘 + 라벨

## 에셋 사용 원칙
- 모든 아이콘과 로고는 `src/assets` 폴더에 제공된 원본 파일(SVG/PNG)만 사용할 것
- 임의로 아이콘 라이브러리(lucide-react, react-icons, heroicons 등)에서 대체 아이콘을 가져오거나 새로 그리지 않음
- 필요한 에셋이 `src/assets` 폴더에 없는 경우, 추측해서 만들지 말고 회색 플레이스홀더 박스로 표시한 뒤 어떤 파일이 필요한지 알려줄 것
- 폴더 구조 예시: `src/assets/icons/` (네비게이션·UI 아이콘), `src/assets/logo/` (로고), `src/assets/images/` (제품 이미지 등 기타 이미지)

## 작업 방식
- 화면 단위로 하나씩 요청 → 컴포넌트 생성 → `npm run dev`로 즉시 확인하는 흐름 유지
- 기존 화면 수정 시 디자인 시스템(컬러/폰트) 임의 변경하지 말고 위 가이드 그대로 적용
- 새로운 컬러가 필요하면 위 팔레트 내에서 가장 가까운 톤을 우선 사용, 임의의 새 색상 추가 지양

## 핵심 화면 (참고용)
- Home: 피부 컨디션 리포트(수분/유분/탄력/피부 점수), 날씨 기반 추천
- Onboarding: 사진 기반 피부 진단 + 라이프스타일 설문 (아래 상세 참고)
- Miyubot: 대화형 AI 챗봇 (제품 탐색, 메이크업 추천, 루틴 가이드)
- Review: 피부 적합도(%), 리뷰어 레벨 시스템(Lv1~Lv5)
- My Beauty / 마이페이지: 피부 데이터 상세, 루틴 관리

### Onboarding 플로우 상세 (4단계)
1. **피부 상태 체크**: 피부 타입(중복선택 칩) + 주요 고민(최대 3개 칩) 선택. 하단에 데이터 이용 동의 안내 박스(연보라 배경).
2. **일상 환경**: 수면시간(라디오 버튼 3개), 음수량(슬라이더 + 숫자 큰 글씨로 표시, Primary 그라디언트 트랙).
3. **얼굴 스캔**: 카메라 풀스크린, 얼굴 위 메쉬/랜드마크 오버레이(보라 톤), 하단 셔터 버튼(Primary 그라디언트 원형).
4. **사진 업로드 안내**: AI 분석 항목 카드(연보라 배경, 아이콘 포함), 정확한 분석을 위한 팁 리스트(아이콘 + 텍스트).

### 공통 컴포넌트 패턴
- **선택형 칩**: 미선택 = GrayScale 70 보더 + GrayScale 40 텍스트 / 선택 = Primary 07 보더 + Primary 07 텍스트, 배경은 흰색 유지
- **하단 고정 버튼 2개**: 좌측 "건너뛰기"(아웃라인, Primary 07 보더+텍스트) / 우측 "다음"(Primary 07 채움, 흰 텍스트), 풀와이드 분할
- **안내/팁 카드**: 연보라 배경(Primary 18~20 또는 GrayScale 90), 라운드 코너, 아이콘+텍스트 조합
- **슬라이더**: 트랙 활성 영역 Primary 그라디언트, 현재 값은 큰 SemiBold 숫자로 강조 표시
- **챗봇(Miyubot) 말풍선**: 단순 텍스트 대화가 아니라, 말풍선 내부에 카드형 UI가 인라인으로 삽입되는 리치 채팅 구조
  - 제품 추천 카드(이미지+이름+가격+추천이유+구매 버튼)
  - 피부 진단 결과 카드(점수, 분석 텍스트)
  - 루틴 단계 카드(번호+제품명+추천 이유)
  - 사용자가 고를 수 있는 빠른 선택 버튼(예: "아침 외출 전" / "저녁 외출 후")도 말풍선 형태로 배치
- **피부 점수 도넛 차트**: 수분/유분/탄력 등 점수를 원형(도넛) 게이지로 시각화, 중앙에 숫자(점) 표시, Primary 그라디언트로 채움
- **제품 상세 페이지**: 상단 메인 이미지는 손으로 제품을 쥔 라이프스타일 컷(단순 누끼컷 X), 피부 적합도(%) 배지, 리뷰 요약 카드
- **리뷰 필터**: 다중 선택 가능 — 별점, 후기유형(사진/영상/텍스트), 피부타입, 피부고민, 피부톤, 사용기간, 연령대, 성별
- **리뷰어 레벨 뱃지**: Lv1(뉴비)~Lv5(마스터) 텍스트 뱃지, 닉네임 옆에 배치, 피부타입·사용기간 메타정보 함께 노출
